#Code Book 

This is the code book for the `tidy.txt` data set produced by the `run_analysis.R` 
script. See the code book for the original data set for more information. 

## Identifiers
* `subject` - Subject ID 
* `activity` - The activity that was performed when the data was collected. See below for the different values.
